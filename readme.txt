各支程式的主要功能
- admin.php : 管理員功能(內含任免管理員、新增看板功能)
- edit_profile.php : 修改個人資料頁面
- index.php : 安安安價讚首頁
- login.php : 登入頁面
- logout.php : 登出頁面
- pdoInc.php : 連接資料庫
- photo.php : 上傳大頭貼頁面
- register.php : 註冊頁面
- viewBoard.php : 瀏覽看板
- viewThread.php : 瀏覽文章與留言
