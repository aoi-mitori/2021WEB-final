# 各支程式的主要用途：
- addBoard.php:新增看板
- admin.php:管理管理員
- edit_profile.php:修改使用者資料
- hw5.php:討論版首頁
- login.php:登入
- logout.php:登出
- pdoInc.php:連接資料庫
- register.php:註冊
- viewBoard.php:瀏覽看板
- viewThread.php:瀏覽主題

# 使用規則
- 標題不可使用骰子
- 樓主可刪除留言

## 骰子類型
請使用半型括號
-  (oj)
-  (queen-rainbow)
-  (dice-six)
