# 各支程式的主要用途：
- addBoard.php:新增看板
- admin.php:管理管理員
- edit_profile.php:修改使用者資料
- index.php:討論版首頁
- login.php:登入
- logout.php:登出
- pdoInc.php:連接資料庫
- register.php:註冊
- viewBoard.php:瀏覽看板
- viewThread.php:瀏覽主題

# 使用規則

- 樓主可刪除該樓留言
